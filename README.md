<h5>Redux logger component</h5>
<p>A react component that provides analytics for the redux actions in your app.</p>
<p>To install</p>

<ul>
    <ol><code>git clone https://github.com/SteliosVoskos/ReduxLoggerComponent.git toYourDir</code></ol>
    <ol><code>cd toYourDir</code></ol>
    <ol><code>npm i</code></ol>
    <ol><code>npm run start</code></ol>
</ul>

<p>Server runs at <code>localhost:300</code></p>
